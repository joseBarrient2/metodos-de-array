ARRAY METHOD

FIND

Busca dentro del arreglo el valor que coincida con la condicion que se le indique. Si encuentra muchas coincidencias solo mostrara el valor de la primera.

SOME

Busca dentro del arreglo si algunos cumplen con la condicion. Si es asi devuelve true y si no devuelve false. Esta no modifica la matriz.

EVERY

Busca dentro del arreglo y evalua si todos cumplen con la condicion que se paso en la funcion, devolviendo true unicamente si todos los elemento del arreglo cumplen, de lo contrario sera false.

MAP

El metodo map evalua cada elemento de un arreglo segun la condicion que se le de, y crea un nuevo arreglo sin alterar el arreglo original.

FILTER

Este metodo creara una copia del arreglo con los elementos que cumplan con los parametros establecidos en la funcion. No modificara el arreglo original.
